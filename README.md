# 🚀 Animated Navigation

A clean and responsive navigation menu with a smooth dark overlay and a changing hamburger icon. It is built completely with CSS and HTML.

### ✨ Features
- Hamburger icon that changes into an "X" when clicked.
- Dark glass effect background using `backdrop-filter: blur`.
- Smooth slide-in animations for the menu links.
- Uses the CSS Checkbox Hack to open and close the menu.

### 🛠️ Built With
- HTML5
- CSS3 (Flexbox, Transitions, and Transforms)

### 💻 How to Run
Just download the files and open `index.html` in your browser.
